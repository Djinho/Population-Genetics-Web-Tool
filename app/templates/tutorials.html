TUTORIALS


Here you can find various tutorials on how to use our analysis tools...


PCA


Select the required populations and press 'analyze' - this will show the PCA results for the selected data. The populations are split into 'superpopulations' and 'populations' allowing the user to distinguish between the two. The 'per sample' button needs to be selected in tandem with the populations to show an sample-centric PCA result.


ADMIXTURE ANALYSIS


Select the required populations and press 'analyze' - this will show the Admixture analysis results for the selected populations, in the format of an interactive barchart side by side. The populations are split into 'superpopulations' and 'populations' allowing the user to distinguish between the two.


SNP ANALYSIS


Select the required populations and the required SNPs in tandem from the list. Press 'analyze' and this will show the relevant statistics, including genotype and allele frequencies, reference and alternate allele, and their clinical relevance. Pressing the 'Calculate FST' will calculate the FST for the selected SNPs, and a pairwise matrix of these. It will also produce a visual representation (a heatmap) of the selected SNPs and a button which allows you to download the results, including the heatmap. There will be a warning if too many SNPs are selected.
